# 基于 react+webrtc+websocket 的仿 bilibili 直播系统

1. react 前端页面
2. webrtc 直播架构
3. socket.io 聊天室

# 本地启动

```bash
yarn start
```

# 环境搭建

```bash
npm -v

```
